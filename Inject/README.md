# 任务书


## 背景

针对针剂瓶表面打印字符在工业生产中出现的断笔划、缺笔画、连笔划等缺陷，以及玻璃表面高光、弧面反射导致的识别准确率下降问题，研究两种缺陷检测方法的可行性：
（1）基于特征点提取与匹配算法的传统图像处理方法；
（2）基于 YOLOv8 深度学习模型 的智能检测方法。
通过构建包含多类缺陷字符的针剂瓶图像训练集，初步验证特征匹配方法的检测准确率，并尝试对比深度学习模型在缺陷识别方面的性能差异，为工业打印字符质量检测提供可行的算法与数据支持。

## 要求

特征匹配算法研究： 设计并实现基于 SIFT、ORB 等算子的特征点提取与匹配算法，探索传统图像处理在针剂瓶打印字符缺陷检测中的可行性；

YOLOv8 缺陷检测： 使用构建的数据集对 YOLOv8 模型进行训练与验证，分析其对不同类型字符缺陷的检测性能；
结果分析与系统实现： 对比特征匹配方法与深度学习模型的检测精度与速度，验证两种方案的适用场景与性能差异，并实现可运行的检测演示系统。

## 路线

1. 开展文献调研与方案设计。查阅针剂瓶字符识别、特征匹配算法与 YOLO 系列模型的相关研究；明确研究目标与技术路线，完成开题报告撰写。

2. 设计并实现特征匹配算法。研究 SIFT、ORB 等特征提取方法，建立针剂瓶字符缺陷检测的传统算法框架，完成算法初步测试。

3. 构建针剂瓶缺陷字符数据集。模拟生成断笔划、缺笔画、连笔划等多类型缺陷字符图像，完成数据增强与标注工作，形成训练与验证集。

4. 进行 YOLOv8 模型训练与性能验证。利用构建的数据集对 YOLOv8 模型进行训练和测试，评估其检测精度、召回率和速度，并与特征匹配方法进行对比分析。

5. 整理实验结果与论文撰写。分析两种检测方法的优缺点，完成论文撰写、图表制作和修改完善，准备论文答辩材料。
